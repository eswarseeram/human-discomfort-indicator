# Human Discomfort Indicator using Arduino

## Overview
This project measures environmental temperature and humidity and calculates the human discomfort index. The system uses an Arduino microcontroller with sensors to determine how comfortable the surrounding environment is and displays the result on an LCD screen.

## Components Used
- Arduino Uno
- Temperature and Humidity Sensor (DHT11)
- 16x2 LCD Display
- Breadboard
- Jumper Wires
- Power Supply

## Working Principle
The DHT11 sensor measures temperature and humidity in the surrounding environment. The Arduino reads the sensor data and calculates the discomfort index based on the measured values.

The calculated discomfort level is displayed on the LCD screen to indicate whether the environment is comfortable or uncomfortable.

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- Discomfort index calculation
- LCD display output

## Project Setup
1. Connect the DHT11 sensor to the Arduino digital input pin.
2. Connect the LCD display to the Arduino.
3. Upload the Arduino code using Arduino IDE.
4. The system continuously reads temperature and humidity values and calculates discomfort level.
